WebGL Liquid Simulator

  1. Do a physics simulation of balls in a container (box2d.js)
  2. Render the simulation onto a raster
  3. Blur the image ()
  4. Apply a threshold ()

Physics simulation is performed by the CPU and the blurring and thresholding is done by the GPU using WebGL. If WebGL is not available you will not see blurring and thresholding.
